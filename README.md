##Tech stack in this project:

1. Front-end

- Next.js
- Redux Toolkit
- Redux DnD
- TailwindCSS
- Rechart/Gantt Chart
- MUI

2. Back-end

- Node.js
- Express.js
- Postgres, PgAdmin
- Postman

3. AWS

- AWS cli
- AWS billing
- AWS EC2 (for hosting backend)
- AWS RDS (for postgreql database)
- AWS Amplify (for hosting next.js)
- AWS S3 (for storage)
- AWS Cognito (for authentication)
- AWS API Gateway
